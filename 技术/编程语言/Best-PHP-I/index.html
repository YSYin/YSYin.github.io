PHP概述

　　PHP，即”PHP: Hypertext Perprocessor”，超文本预处理器，是一种被广泛应用的开源通用脚本语言，尤其适用于Web开发并可嵌入HTML之中。与使用C或Perl语言通过大量命令编写程序以输出HTML不同的是，==PHP页面就是HTML==，只不过在其中嵌入一些代码进行处理，PHP代码被包含在特殊的起始符和结束符<?php和?>中。

　　和客户端的javascript语言不同，**==PHP代码运行在服务器端==，可以使用PHP完成任何其他的CGI程序能够完成的工作，如收集表单数据、生成动态网页、发送/接收Cookies等等。在服务器上安装PHP解析器（CGI或者服务器模块）**后，只需要建立.php文件，并将其放置到web目录中，在客户端访问.php文件时，服务器会自动通过PHP解析器解析PHP文件并生成HTML。

　　PHP除了可作为服务器端脚本外，还也用于：

  • 命令行脚本。不通过服务器或浏览器运行，仅通过PHP解析器执行
  • 编写桌面应用程序。通过PHP-GTK编写具有图形界面的桌面应用程序。

基本语法

PHP标记

解析PHP文件时，解析器寻找PHP起始符和结束符，以确定何处开始和停止解析代码，任何起始符和结束符之外的部分都会被PHP解析器忽略（直接输出）。

  • <?php和?>：官方推荐的PHP标准标记，所有版本/类型解析器均可识别
  • <?和?>：允许但不推荐，需要激活php.ini中的short_open_tag配置或者在编译PHP时使用配置选项--enable-short-tags

注意，如果文件内容是纯PHP代码，则最好在文件末尾删除PHP结束标记，以避免PHP结束标记后意外加入空格或者换行符，进而导致解析器输出这些空白

从HTML中分离

凡是PHP开始标记和结束标记之外的内容都会被解析器忽略，因此PHP文件可以具备混合内容；亦可以使PHP嵌入到HTML文档（此时文档名必须以.php为后缀）中，但有两个特殊情况：

  • 结束标记后紧跟换行：该换行不会被输出

  • 内容位于条件语句中间：PHP解析器根据条件判断哪些输出、哪些跳过

    • <?php if ($expression == true): ?>
  This will show if the expression is true.
<?php else: ?>
  Otherwise this will show.
<?php endif; ?>

    • 要输出大段文本时，跳出PHP解析模式通常比将其包含在PHP代码之中使用echo/print输出更有效率

指令分隔符

PHP需要在每个语句后用分号结束指令，两种例外：

  • PHP结束标记?>隐含表示一个分号，故其之后不必再加分号
  • PHP代码段中的最后一行可以不用分号结束

注释

  • 单行注释：//...和#...，注释到行末或者当前的PHP代码块结束
  • 多行注释：/**/，碰到第一个*/结束

类型

简介

PHP支持9种原始数据类型

  • 四种标量类型
    • boolean（bool，布尔型）
    • integer（int，整型）
    • float（浮点型，也称作double）
    • string（字符串）
  • 三种复合类型
    • array（数组）
    • object（对象）
    • callable（可调用）
  • 两种特殊类型
    • resource（资源）
    • NULL（无类型）

注意：

  • **==变量的类型通常不是由程序员设定的，是由PHP解析器根据该变量使用的上下文在运行时决定的==**（解释型语言的特点）
  • 可使用var_dump()函数查看某个表达式的值和类型、使用gettype()得到表达式的类型，使用is_type(): is_int()/is_str()检验表达式是不是某个类型

Boolean布尔类型

两个真值：True/False，不区分大小写

类型转换：

  • 自动类型转换
  • 强制类型转换：(boolean)或(bool)

自动或强制转换为boolean时，以下值被认为是FALSE：

  • 布尔值False本身
  • 整型值0与-0
  • 浮点型0.0与-0.0
  • 空字符串，以及字符串”0”
  • 不包括任何元素的数组
  • 特殊类型NULL（包括尚未赋值的变量）
  • 从空标记生成的SimpleXML对象

所有其他值被认为是TRUE

Integer整型

整型值可使用十进制、十六进制、八进制、二进制表示：

  • 八进制数字前加上0
  • 十六进制数字前加上0x
  • 二进制数字前加上0b
  • PHP 7.4.0开始，整型数值可包含下划线，便于阅读

特殊值：

  • 字长使用PHP_INT_SIZE表示
  • 最大值PHP_INT_MAX
  • 最小值PHP_INT_MIN

溢出

  • 给定的数超过integer的范围时，将会自动转换为float

取整

  • 无整除运算符，结果为浮点型
  • 可通过强制类型转换、round函数、intdiv函数进行取整

类型转换

  • 自动类型转换
  • 强制类型转换(int)和(integer)和intval()
